# Graph Generator for Flow Cytometry 
**Background**
Commercial flow cytometry software (e.g., FlowJo) is widely used for analyzing .fcs files and generating publication-ready figures. However, these tools operate as black boxes, the underlying calculations and normalization steps are largely opaque, making reproducibility and transparency difficult to ensure. Additionally, their rigid interfaces offer limited flexibility for customizing figure elements to fit specific research needs.

**Solution**
This tool was developed to address those limitations by providing an open, Python-based alternative for processing and visualizing flow cytometry data. Built on the Plotly library, it gives users full visibility into every calculation and normalization step applied to their samples, while enabling extensive customization of all figure elements, from axis scaling to color schemes and layout.

**Key Features**
* Full transparency into data processing and normalization logic

* Interactive, publication-quality figures via Plotly

* Flexible customization of all graph elements

* Open-source and reproducible, no black-box calculations
